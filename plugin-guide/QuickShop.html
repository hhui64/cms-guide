# 快捷商店

快捷商店由 QuickShop 插件提供，本服务器所使用的插件版本由 CSSBase64 魔改，点击这里浏览魔改版本的源码仓库。

# 创建商店

WARNING

创建商店需要收取 500 游戏币 的创建费用，这笔费用在你移除商店时将会返还至你的账户中。

手持物品，按住Shift+左键单击任意一个箱子，然后根据提示完成操作。

# 移除商店

将商店箱子破坏即可

# 交易税

每笔交易将会收取 0.05% 的交易税。
